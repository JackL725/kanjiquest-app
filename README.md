# KanjiQuest 漢字クエスト

A gaming-specific Japanese kanji SRS app. Study kanji from your favourite JRPGs before you play them.

## Stack
- React + Vite
- Tailwind CSS (Ink & Steel design system)
- React Router
- SM-2 Spaced Repetition Algorithm

## Design system
- **Fonts:** Crimson Pro (display) · DM Sans (body) · IBM Plex Mono (mono) · Zen Kaku Gothic New (kanji)
- **Colors:** ink (backgrounds) · gold (accents) · parchment (text) · ember (danger/again)

## Screens
- `/library` — home dashboard, owned decks, due cards
- `/browse` — deck store
- `/deck/:id` — deck detail + card list
- `/study/:id` — SRS study session
- `/radicals` — primitive & radical reference
- `/profile` — user profile + achievements

## Getting started
```bash
npm install
npm run dev
```
